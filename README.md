# SBG Multi Goals

Adds a lobby rule Goals Per Hole (1-100).
If set above 1, a player must score N times in the same hole before finishing.
Adds a kind of comeback mechanic and a way to extend the gameplay on a single map.

## Install
Unzip into game folder so files land under BepInEx/plugins/SBGMultiGoals/MultiGoals.dll.

## Notes
Host-only install required
